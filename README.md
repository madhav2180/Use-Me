# Use-Me
A cleaner and smarter waste management system using IoT sensors using AI &amp; ML.



## How to configure
**1.** Clone git repo

```shell
$ git clone "https://github.com/madhav2180/Use-Me.git"
```

**2.** Create virtual env

```shell
$ pip install virtualenv
```
```shell
$  python -m venv env
```
```shell
$  ./env/Scripts/activate
```
